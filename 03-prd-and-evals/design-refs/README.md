# Design References - Marginalia TV

## Reference Products

- **Prime Video X-Ray** - borrow the intent of contextual story assistance; avoid forced overlays and spoiler-prone metadata.
- **Trakt** - borrow clear show/season/episode hierarchy and progress language.
- **Linear** - borrow quiet information density, precise states, and restrained interaction patterns.
- **Notion** - borrow readable structured blocks for status cards, tables, and checklists.

## Borrow List

- Persistent context banner that states the exact boundary.
- Dense but readable dashboard showing in-progress shows.
- Typed relationship/status rows that can be scanned without reading an essay.
- Compact tables for "Secrets and Who Knows Them."
- Checklist pattern for "Ready to Watch."
- Muted professional palette that feels like a serious PM prototype, not a fan site.

## Avoid List

- Purple primary buttons.
- Gradient backgrounds.
- Emoji-heavy states.
- shadcn default styling without strong customization.
- "Acme Corp," "Show 1," "Character A," or Lorem ipsum.
- Future-season thumbnails, trailer stills, promotional images, or episode titles beyond the selected boundary. For the MVP, no Season 5 imagery or titles.
- "Powered by AI" badges, sparkle icons, or AI-first marketing language.
- Loading skeletons or empty states; the demo should render instantly with seeded data.
